# 💾 Лабораторна робота №12–13  
## Тема: Робота з файлами в C++

---

## 🎯 Мета роботи
Навчитись створювати, зчитувати, записувати та аналізувати дані у текстових файлах за допомогою мови програмування C++,  
а також використовувати структури та масиви для збереження й обробки інформації.

---

## 🧩 Хід роботи

### 🧮 Завдання 1 — Робота з числами у файлі
Умова:  
Створити текстовий файл number.txt, який містить 10 чисел.  
Програма повинна зчитати ці числа, обчислити суму, середнє значення, знайти максимальне та мінімальне число,  
а також вивести результати на екран.

Файли завдання:  
- main.cpp — програма для зчитування й обробки чисел  
- number.txt — вхідний файл із числами  
- README.md — опис завдання, алгоритму та приклад виконання  

---

### 🧮 Завдання 2 — Робота з двовимірним масивом (матрицею)
Умова:  
Створити текстовий файл matrix.txt, який містить числові дані у вигляді матриці.  
Програма має зчитати дані, зберегти їх у двовимірний масив,  
та виконати обробку — наприклад, підрахунок суми елементів, середнього значення тощо.  
Результати відобразити у зручному вигляді на екрані.

Файли завдання:  
- main.cpp — програма для зчитування та обробки матриці  
- matrix.txt — файл із даними  
- README.md — опис алгоритму роботи програми  

---

### 👩‍🎓 Завдання 3 — Робота зі структурами
Умова:  
Реалізувати програму, яка демонструє роботу зі структурами у C++.  
Створити структуру Grades для оцінок і структуру Student для зберігання імені, курсу та оцінок.  
Програма вводить дані кількох студентів і виводить тих, у кого всі оцінки знаходяться в межах 75–100 балів.

Файли завдання:  
- main.cpp — основний код програми  
- README.md — опис логіки програми та приклад результату
