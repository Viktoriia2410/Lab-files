# 🧮 Завдання 1 — Робота з числами у файлі

## 📖 Опис завдання
Створити програму, яка зчитує 10 чисел із текстового файлу number.txt,  
виконує обчислення суми, середнього арифметичного, а також визначає  
мінімальне та максимальне значення.  
Отримані результати потрібно вивести на екран.

---

## 🧠 Алгоритм виконання
1. Відкрити файл number.txt для читання за допомогою ifstream.
2. Зчитати всі числа у змінні або масив.
3. Під час зчитування:
   - додавати кожне число до загальної суми;
   - перевіряти, чи є воно більшим або меншим за поточні максимальне та мінімальне значення.
4. Після завершення зчитування:
   - обчислити середнє арифметичне;
   - вивести на екран кількість чисел, суму, середнє, мінімальне і максимальне значення.
5. Закрити файл після завершення обробки.
