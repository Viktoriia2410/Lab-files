# 🧮 Завдання 2 — Робота з двовимірним масивом (матрицею)

## 📖 Опис завдання
Необхідно створити програму, яка зчитує з текстового файлу matrix.txt двовимірний масив (матрицю) чисел  
та виконує базову обробку даних у пам’яті: заповнення, підрахунок сум, середніх тощо.

---

## 🧠 Алгоритм роботи з двовимірним масивом

1. Відкрити файл `matrix.txt` за допомогою об’єкта ifstream.
2. Зчитати розмірність матриці (кількість рядків і стовпців).
3. Створити двовимірний масив відповідного розміру.
4. Зчитати елементи матриці з файлу та записати їх у масив.
5. Обробити дані — наприклад, знайти суму або середнє значення елементів.
6. Вивести результати обробки на екран у зручному вигляді.
7. Закрити файл після завершення зчитування.
