# 👩‍🎓 Завдання 3 — Робота зі структурами в C++

## 📖 Опис завдання
Необхідно створити програму, яка демонструє роботу зі структурами в мові C++.  
Програма опрацьовує дані про студентів — їх імена, курс та оцінки з трьох предметів.  
Мета — знайти та вивести лише тих студентів, у яких усі оцінки знаходяться в межах від 75 до 100 балів.

---

## 🧠 Логіка роботи програми

1. Оголошуються дві структури:
   - Grades — зберігає оцінки з трьох предметів.  
   - Student — містить ім’я, курс та структуру Grades.
2. Користувач вводить кількість студентів.  
3. Для кожного студента вводяться:
   - ім’я  
   - курс  
   - три оцінки  
4. Програма перевіряє, чи всі оцінки знаходяться в діапазоні 75–100.  
5. Якщо так — дані студента виводяться на екран.  
6. Якщо жоден студент не відповідає умові, виводиться повідомлення про відсутність таких студентів.
